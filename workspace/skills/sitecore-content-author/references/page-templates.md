## Most Common Templates

### Landing Page (Default)
- **Template ID**: `300f3d1b-52ef-4734-8eab-ae2e2a422759`
- **Use for**: Most general pages, marketing pages, product pages
- **Supports**: Full component flexibility

### Article Listing Page
- **Template ID**: `7a6105a8-f696-4a58-865a-510e8998956b`
- **Use for**: Blog index, news index, article archives
- **Supports**: Filtering, pagination, search

### Event Listing Page
- **Template ID**: `959f1c50-f8f6-4c59-9cd0-2f8f21301835`
- **Use for**: Event calendars, webinar listings
- **Supports**: Date filtering, calendar views

### Error Page
- **Template ID**: `13db779e-4ae1-44f9-b83d-1500ef844adc`
- **Use for**: 404 Not Found, 500 Server Error pages
- **Supports**: Basic content components

## Usage Notes

- **No MCP needed**: Use this file instead of `mcp__marketer-mcp__list_available_insertoptions`
- **Token savings**: ~600 tokens per lookup
- **Default choice**: Use Landing Page for most new pages
- **Parent context**: These templates are available under `/sitecore/content/sites/main/home`
